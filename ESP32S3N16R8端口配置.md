# ESP32S3N16R8 芯片端口配置

## 1. 基本信息
- **芯片型号**：ESP32S3N16R8
- **核心频率**：最高240MHz
- **内存**：16MB Flash，512KB SRAM
- **GPIO数量**：49个

## 2. 项目端口配置

### 2.1 电机驱动端口
| 功能 | 引脚 | 说明 |
|------|------|------|
| 电机AIN1 | GPIO 18 | 电机A方向控制1 |
| 电机AIN2 | GPIO 19 | 电机A方向控制2 |
| 电机BIN1 | GPIO 21 | 电机B方向控制1 |
| 电机BIN2 | GPIO 47 | 电机B方向控制2 |
| 电机PWMA | GPIO 9 | 电机A速度控制（PWM） |
| 电机PWMB | GPIO 10 | 电机B速度控制（PWM） |

### 2.2 舵机驱动端口
| 功能 | 引脚 | 说明 |
|------|------|------|
| 舵机1 | GPIO 22 | 舵机1控制（PWM） |
| 舵机2 | GPIO 23 | 舵机2控制（PWM） |

## 3. 硬件连接说明

### 3.1 电机驱动连接
- 使用TB6612或类似的电机驱动芯片
- 方向引脚（AIN1, AIN2, BIN1, BIN2）连接到电机驱动芯片的方向控制端
- 速度引脚（PWMA, PWMB）连接到电机驱动芯片的PWM控制端

### 3.2 舵机连接
- 舵机1和舵机2的信号端分别连接到GPIO 22和GPIO 23
- 舵机的VCC和GND需要连接到适当的电源（通常为5V）

### 3.3 OLED连接
- 使用I2C通信协议
- SDA和SCL引脚需要连接上拉电阻（通常为4.7KΩ）
- OLED的VCC和GND需要连接到适当的电源（通常为3.3V）

## 4. PWM配置

### 4.1 电机PWM配置
- 频率：1000Hz
- 分辨率：8位（0-255）
- 通道：LEDC_CHANNEL_0 (PWMA), LEDC_CHANNEL_1 (PWMB)
- 定时器：LEDC_TIMER_0

### 4.2 舵机PWM配置
- 频率：50Hz
- 分辨率：10位（0-1023）
- 通道：LEDC_CHANNEL_2 (舵机1), LEDC_CHANNEL_3 (舵机2)
- 定时器：LEDC_TIMER_1

## 5. 代码配置

### 5.1 电机初始化
```cpp
// 在mcp_server.cc中
if (g_motor == nullptr) {
    g_motor = new MotorDriver(18, 19, 21, 47, 9, 10);
    g_motor->init();
}
```

### 5.2 舵机初始化
```cpp
// 在mcp_server.cc中
if (g_servo == nullptr) {
    g_servo = new ServoDriver(22, 23);
    g_servo->init();
}
```

### 5.3 OLED初始化
```cpp
// 在application.cc中
OLED_Init();
// 显示圆形
OLED_DrawCircle(64, 32, 18, OLED_FILLED);
OLED_Update();
```

## 6. 注意事项
- 确保所有GPIO引脚的电流不超过ESP32S3的最大输出电流（每个GPIO最大为40mA）
- 舵机和电机需要外部电源供电，不能直接从ESP32S3获取
- I2C通信需要上拉电阻以确保稳定通信
- PWM通道和定时器的配置应避免冲突
